# Hammerwold

Hammerwold is an industrial platform designed to connect companies, contractors, and workers for executing large-scale industrial projects efficiently.

## Features

- **User Signup & Login**: Register and log in as a company, contractor, or worker.
- **Post Projects**: Companies can post projects with requirements and invite contractors to bid.
- **Browse Projects**: Contractors can browse available projects and submit bids.
- **Direct Worker Hiring**: Workers can build their profiles and get hired by companies and contractors.

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL (using Neon)
- **Deployment**: Vercel
